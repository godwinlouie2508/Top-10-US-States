# Top 10 US States Dashboard

An interactive Excel dashboard analyzing and visualizing the top 10 US states across five critical socioeconomic parameters: GDP, Cost of Living, Healthcare, Population, and Crime rates.

## üìä Overview

This project provides a comprehensive, user-friendly Excel dashboard that enables data-driven comparison of US states. Built with advanced Excel features including VBA macros, dynamic slicers, and interactive visualizations.

## ‚ú® Features

### Interactive Navigation
- **Central Home Screen**: Navigate between five parameter sheets using VBA-powered buttons
- **Automated Workbook Opening**: Macro automatically selects Home sheet on startup
- **Info Icon**: Toggle instructions with show/hide functionality

### Dynamic Data Filtering
- **Slicers**: Easily filter and select top 10 states across GDP, Cost of Living, Healthcare, and Crime parameters
- **Dropdown Menus**: Display detailed statistics and ranks for individual states outside the top 10
- **Data Validation**: Input ranks (1-50) on Population sheet to view corresponding state data

### Rich Visualizations
- **Bar Charts**: GDP comparison
- **Pie Chart**: Cost of Living breakdown
- **Column Chart**: Healthcare metrics
- **Map Chart**: Population distribution
- **Stacked Column Chart**: Crime statistics

## üöÄ Getting Started

### Prerequisites
- Microsoft Excel 2016 or later (for XLOOKUP support)
- Macros enabled in Excel

### Installation
1. Download the Excel workbook file
2. Enable macros when prompted
3. The Home sheet will automatically open

### Usage
1. **Navigation**: Click parameter buttons on Home sheet to view different analyses
2. **Filtering**: Use slicers to select specific states for comparison
3. **Detailed View**: Use dropdown menus to view statistics for any state (ranks 1-50)
4. **Help**: Click the info icon for navigation instructions

## üîß VBA Macros Implemented

- `Workbook_Open`: Auto-selects Home sheet on workbook open
- Navigation macros: Switches between parameter sheets
- `InfoShow`: Displays navigation instructions
- `InfoHide`: Dismisses instruction overlay
